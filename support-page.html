How to Use CineFormat AI

Open a Document

Open any Google Docs document where you would like to format a screenplay.

Open the Sidebar

Go to Extensions → Screenplay Formatter → Open Sidebar from the Google Docs menu bar.

Convert Text

Paste unformatted text into the sidebar and click Convert to Screenplay, or select text in the document and choose Convert Selected Text from the add-on menu.

Review Output

The add-on uses the Google Gemini API to process your text. The formatted result is inserted directly into your document.

Troubleshooting

I clicked Convert but nothing happened

Make sure you have an active internet connection. The add-on requires internet access to communicate with the Google Gemini API. Also verify that you granted all required permissions when you first opened the add-on.

I see a Gemini API error

If you encounter an error related to the Gemini API, ensure that a valid API key has been configured. This is a developer-level requirement set up during installation.

The formatting doesn't look right

Review the output and use the sidebar's custom styling options to adjust font, size, color, or indentation for specific paragraphs. For best results, ensure your input text clearly separates screenplay elements — scene headings, dialogue, action lines — on separate lines.

The add-on doesn't appear in the Extensions menu

Try refreshing your Google Docs page. If the add-on still does not appear, uninstall and reinstall it from the Google Workspace Marketplace.
